Who are we?

We currently are a very small team aiming at the development of Open Source software. Our first true project will have been Typee Language Project which is presented in this Web site and for which Open Source code can be found and downloaded from the related GitHub repository https://github.com/Typee-Language/Typee. The only thing we expect is that our developments will benefit the software development community.  

Why this .ovh domain?

Well, Typee is an Open Source Project and this site is not for any commercial use. So, we had to do things as cheaper as possible. OVH is the host company for our Web site. OVH is a French company which proposes domain names with their company name as extension for the cheapest price we have been able to find according to the high level of the services they provide. Trust us, this was far cheaper than .fr, .com, .eu, .net, .org or even .bzh extensions (did you say "bzh"? Yes. But only bretons can understand this).  

Finally

We are based in France, in a small village of Britanny, and we are software development warriors.
